**James Clear**的《**原子习惯**》（**Atomic Habits**）尝试帮助人们养成自己想要的习惯。

从我们呱呱落地的那一天开始，我们就从未停止过“养成习惯”。我们学会了按时吃饭，随手关门...乍一听，养成习惯是一件稀松平常、理所应当的事情。

但当我们想要主动培养一个习惯时，却意外地发现困难重重。想学一门乐器？想每天坚持运动？开始阅读？你一定尝试过类似的事情。如果结局常常以失败告终：

首先，请不要陷入自我审判。

其次，这里是15条书中让我受益匪浅的建议：

***

# 1. 带入角色

如果你把自己带入某一个角色，那么这个角色会做的事情，就是你会做的事情。例如如果你想学弹钢琴

`我是一个音乐家，而音乐家每天都会弹琴` 

相比单纯的

`我每天都要弹琴` 

要更有说服力。

> 我认同自己软件工程师的身份，因此听播客和做业余项目对我而言是理所当然的。

> 我的游戏开发之旅也是从\"假装自己是一名游戏开发者\"开始的。远在我第一次打开游戏引擎之前，我常常问自己：\"如果我是一名游戏开发，此时此刻我会做什么、想什么？\"。在我看来这和硅谷的\"Fake it till you make it\"有异曲同工之妙。

# 2. 你不需要提升100%
如果你把你的生活分成100个部分，然后在每个部分中只提升1%，你最终还是会积累巨大、巨大的进步。

# 3. 喊出来
大声、明确地告诉自己要做什么。

`我每天午休去吃饭之前，我会合上电脑，然后做10个俯卧撑` 

比

`我每天要运动`

更加明确，所以更有可能被执行。

# 4. 自控力

自控力最强的人往往最不需要使用自控力。你其实可以避免把自己放在充满诱惑的环境里。

> 我控制自己不吃薯片的方式是压根不买薯片。在超市里跳过薯片区，比在夜深人静的时候不去想薯片，要容易得多。

# 5. 捆绑欲望
将你想要培养的习惯与你已经养成（并且享受！）的习惯捆绑在一起。

> 我一直都想自己做早餐，但从来没有付诸实践，直到我拥有了一台半自动咖啡机 - 我爱做咖啡和拉花。早起变成了一件令人兴奋的事情。至于早餐？只是顺手做的而已;D

# 6. 环境
* 你能直接看见什么很重要。试试在枕头上放一本你想读的书，把你的琴放在一进屋就能看见的地方...
* 我们会无意中模仿我们仰慕的人，试试多和他们呆在一起
* 一件事情对其他人来说可以是不寻常的，但只要你把它变成你的生活中的\"常规\"行为，你就会很容易爱上它 (搜搜看，laszlo家族和国际象棋)
* 尝试和那些已经和你有共同点的人一起培养习惯，会更容易

# 7. 让它够简单
你可以以最小的步长去推进自己的习惯，走得慢有时反而走得远。

第一个好处是容易坚持。你不需要每天跑步30分钟来养成跑步的习惯，你只需要让自己跑1分钟。谁都可以做到跑1分钟，不是吗？在任何动作成为负担前停止 - 毕竟这不是工作！

第二个好处是更精妙的 - 饥饿营销。也许你只有3分钟热度，但如果你只允许自己跑1分钟，你就保住了2分钟的热情。如果每次你做一件事情时你的热情都不是被透支而是产生了盈余，你做这件事情的热情就会越来越高涨。

> 我说服自己每天弹琴的方式是计时3个10分钟。10分钟到了，即使我还想继续弹当时的练习/曲子，我也逼迫自己停止。我总是意犹未尽地离开我的电子琴。

# 8. 立竿见影
捆绑长期收益和即时满足。人类总是倾向于放大即时满足（长期主义者也一样！）。

# 9. 让它够爽
特别是当你完成它的时候！和训狗狗一样，当你奖励一个行为，这个行为就会被重复；当你惩罚一个行为，这个行为就会被避免。

# 10. 只允许一次例外
第一次错过是意外，第二次错过是新（不想要）的习惯的开始。

# 11. 追踪你的习惯
我们都喜欢看到进展。

# 12. 但是...
当一个指标变成目标，它可能就不再是一个好的指标。

# 13. 持续突破
当你感觉习惯基本成型后，你应该尝试持续突破。给自己新的挑战，并且享受战胜它带来的喜悦。一个喜剧演员可以每年都给自己增加几分钟的台词，若干年之后他就能有自己的脱口秀了。

# 14. 关于无聊
* 事实是，没有人有无限的热忱和激情
* 你必须为你的习惯的养成过程创造出新颖的元素
* 专业人士会坚持执行计划，不管情绪如何（就算他感到烦躁或无聊）

# 15. 三省吾身
* 习惯的暗面是思维惯性;当你做一件事情太过于熟练时，你可能就不再注意到一些可供提升的细节了
* 你应该保持你的（养成习惯时所做的）付出，而不是仅仅是重复你已经养成好的习惯
* 投资者会做决策日志，运动员会记录训练，喜剧演员会测试笑话...你也应该对自己习惯的事做类似的事情
